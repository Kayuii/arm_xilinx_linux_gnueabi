Sourcery Probe Personal Network Ports

The Sourcery Probe Personal models for ARM and MIPS use remote RNDIS for TCP/IP communication to Windows over USB, and CDC/Ethernet for TCP/IP communication to Linux over USB. Specifically, the following network ports are used:

Table A-1. Sourcery Probe Personal Network Ports

Port Number

Description

19119

Used by ARM and MIPS debuggers for probe configuration

51410

Used by ARM and MIPS debuggers to control the probe
