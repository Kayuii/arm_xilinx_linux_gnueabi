Mentor Embedded Sourcery Probe Professional (Sourcery Probe Professional) is a tool that helps you develop and debug embedded software.

 

Tip: If you have a probe that is labeled Mentor Embedded GIGA-JTAG Probe, this works exactly the same as Sourcery Probe Professional. The only difference is the label on the hardware. Follow the instructions in the manual for Sourcery Probe Professional.
