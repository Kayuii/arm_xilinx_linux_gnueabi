KA (Kill Alias)

Availability: ALL

The KA (Kill Alias) command deletes the name and replacement text for one or all of the currently defined command aliases. Command aliases are created with the Enter Alias command. See EA (Enter Alias) for more information.

Usage

KA *

KA ident

Arguments

  • *
Remove all aliases.
  • ident
The name of a specific alias to remove. ident is not case sensitive.
