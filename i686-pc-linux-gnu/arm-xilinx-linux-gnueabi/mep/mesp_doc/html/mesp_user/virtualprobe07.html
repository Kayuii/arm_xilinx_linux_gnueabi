Options for Ending a Debug Session

The way you end a debug session depends on how the virtual probe was started.

  • If the CodeBench debugger automatically started the virtual probe and sprite server, then it will automatically close them at the end of a debug session.
  • If the software side of the testbench launched the virtual probe and sprite server, then see the Veloce Virtual Probe User Guide for instructions on closing them.
  • If vmajic was run manually, then pressing <ESC> in its console window brings up the setup menu, from where you can choose to disconnect and quit, or shut down emulation and quit.
