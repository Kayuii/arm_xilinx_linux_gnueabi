Ending a Debug Session

  • Terminate debugging in CodeBench. This ends the debug session, but leaves the emulation job and vmajic debug server still running, so you can relaunch the debugger for a new session.
  • Press <ESC> in the MESP/Virtual console window and choose 'S' to shut down the emulation job and vmajic debug server.
Note

Setting the Mentor Embedded Sourcery Probe Ice_Shutdown_On_Exit configuration option to yes allows you to automatically shut down vmajic and the Veloce emulation job when terminating the debug session. See Table 3-9 for details.
