DO (Display Options)

Availability: ALL

The DO (Display Options) command displays a table of specified configuration options with their current value and a brief description. If the command is entered without parameters, all options are displayed. The Display Options Verbosely (DOV) command fully describes the option or options; it is best used with a specific option. The Display Options Quietly (DOQ) command displays only the option's name and current value. To modify configuration options, see the EO (Enter Option).

Usage

DO[Q|V] [*|pattern*|cfg_option]

Arguments

  • *
Displays all options. This is the default. ‘*’ is a wildcard character and can be given alone or at the end of pattern.
  • pattern*
Displays all configuration options where the first letters match pattern.
  • cfg_option
Displays a specific option. The various options available are described in Configuration Options.
